# BFSI AI Assistant

AI-powered Banking Customer Support Assistant built using NLP and Streamlit.

## Features
- BFSI dataset based responses
- Semantic similarity search
- LLM fallback model
- Streamlit chatbot UI

## Tech Stack
Python  
Transformers  
Sentence Transformers  
Streamlit  

## Run Locally

pip install -r requirements.txt  
streamlit run app.py
